IonicX Desktop

Plataforma de presentaciones en vivo para iglesias: Biblia, letras de canciones, portadas de prédicas y salida en pantalla completa.

Descargar Última versión

Qué puedes hacer

Biblia al instante

Busca versículos y envíalos al modo en vivo con un clic.

Letras de canciones

Edita, segmenta y controla escenas para cada parte de la canción.

Portadas de prédicas

Crea diseños con capas, badges e imágenes en segundos.

Proyección

Salida “External Live” a pantalla completa para pantallas externas.

Instalación

Windows

Ejecuta el instalador .msi y sigue el asistente.

macOS

Abre el .dmg, arrastra IonicX a Aplicaciones y ejecútalo.

Linux

Usa el .AppImage para abrir rápido, o instala el paquete .deb/.rpm.
